## Included

* Dotnet sdk 3.1
* Dotnet Sonar Scanner

## Build image

```bash
docker build -t izapelini/dotnet-sdk-ci .
```

## Run docker

```bash
docker run --rm -it izapelini/dotnet-sdk-ci bash
```
